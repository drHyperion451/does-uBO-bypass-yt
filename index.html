Still having an annoying popup? See r/uBlockOrigin

What does this mean?

It means that uBlock Origin has blocked the Anti-Adblocker script and now you can safely watch youtube without ads

What should I do now

You need to delete all cached ublock filters and update them. There is a good guide made by the ublock origin team themselves in Reddit.

What does this page do?

It simply gets the info of the last uBlock fix ID and the youtube Anti-Adblocker script ID. If it's the same, then the uBlock team has finally updated their filters. If it's not, a fix is on the way.
